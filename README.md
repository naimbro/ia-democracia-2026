## Como editar

- Editar `index.html` directamente.
- Mantener HTML, CSS y JavaScript vanilla; no hay build step.
- Usar enlaces relativos para archivos de este repo y `target="_blank"` para enlaces externos.
- Desplegar cambios haciendo push a `main`; GitHub Pages publica desde la raiz del repo.
